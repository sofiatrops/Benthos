# CONTEXTO DEL PROYECTO

Benthos es una empresa especializada en ingeniería en gestión ambiental.

La empresa realiza:

* Monitoreo de calidad de agua.
* Estudios de macroinvertebrados.
* Estudios de microinvertebrados.
* Gestión de campañas ambientales.
* Generación de informes técnicos.
* Trabajo con salmoneras.
* Trabajo futuro con mineras.
* Trabajo futuro con hidroeléctricas.
* Trabajo futuro con organismos públicos.

El sistema será una PLATAFORMA WEB SaaS.

No será una aplicación móvil.

No será una aplicación de escritorio.

Toda la solución debe estar diseñada para ejecutarse como una plataforma web moderna.
